## Muloqot

- Foydalanuvchi bilan **o'zbek tilida** muloqot qil.
- Saytdagi/botdagi barcha o'zbekcha matnlar **adabiy tilda** bo'lsin: «buyurtma» (zakaz emas),
  «yuklab olish» (skachat emas), «sozlamalar» (nastroyka emas). Ruscha o'zlashma so'z ishlatma.

## Ishlash tartibi

- O'zgarishdan keyin `npm run build` va `npm run lint` o'tkaz.
- UI o'zgarishlarini brauzerda skrinshot bilan tekshir (dev server: `npm run dev`).
- `git push` production'ga avtomatik deploy qiladi (Vercel) — push'dan oldin build o'tganiga ishonch hosil qil.
- `.env.local`dagi kalitlarni hech qachon chatga chiqarma va commit qilma.
